# 🚀 Full-Stack Todo App (Streamlit + Nhost + GraphQL)

> A full-stack task management application demonstrating real-time CRUD operations using a GraphQL backend and Streamlit frontend.

---

## ⚡ Overview

This project showcases how to build a **modern full-stack application** by integrating a lightweight frontend with a backend-as-a-service platform.

It uses **Nhost (PostgreSQL + GraphQL)** to handle data operations and **Streamlit** for rapid UI development.

👉 Focus: **API integration, backend connectivity, and real-time data operations**

---

## 🧠 Key Features

* ➕ Create new tasks
* 📋 Fetch and display tasks (GraphQL queries)
* ✅ Update task status (mark complete)
* ❌ Delete tasks
* 🔄 Real-time backend interaction

---

## 🏗️ System Architecture

```bash
Streamlit Frontend
        │
        ▼
GraphQL API (Nhost)
        │
        ▼
PostgreSQL Database
```

👉 Demonstrates **frontend ↔ API ↔ database flow**

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Nhost (PostgreSQL + GraphQL API)
* **Query Language:** GraphQL
