# 📝 Todo App using Nhost + Streamlit

## 🚀 Overview
This is a simple Todo Application built using:

- **Frontend:** Streamlit
- **Backend:** Nhost (PostgreSQL + GraphQL)

The app demonstrates full CRUD operations:
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks
